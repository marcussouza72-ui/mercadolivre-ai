# MercadoLivre AI SaaS

Plataforma SaaS para vendedores profissionais do Mercado Livre com foco em automação, inteligência artificial, análise de dados e gestão operacional.

---

# Status do Projeto

**Versão:** 1.0.0

**Fase Atual:** Foundation ✅ Concluída

Próxima etapa:

➡️ Integração completa com a API do Mercado Livre.

---

# Principais Tecnologias

## Backend

- Python 3.12
- FastAPI
- SQLAlchemy 2
- PostgreSQL
- Alembic

## Segurança

- JWT
- Refresh Token Rotation
- Argon2 (pwdlib)

## Arquitetura

- Repository Pattern
- Service Layer
- Dependency Injection
- Arquitetura Modular
